Nebula Release Package

This archive contains:

- `nb`
- `nbtui`
- install script for your platform

Run directly:

Linux/macOS:

```bash
./nb
./nbtui
```

Windows:

```powershell
.\nb.exe
.\nbtui.exe
```

Install globally:

Linux/macOS:

```bash
sudo mv nb /usr/local/bin/
sudo mv nbtui /usr/local/bin/
```

Windows:

1. Move `nb.exe` and `nbtui.exe` to a folder you keep for tools.
2. Add that folder to your `PATH`.

Or use the included install script for your platform.
